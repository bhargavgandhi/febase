# FE Base

## Technology Used

HTML5
CSS3 ( SASS )
jQuery
Javascript
Bootstrap
FontAwesome Icons (CDN source from bootstrap)
Google Maps Javascript API

html5shiv.min.js - For HTML5 elements support in IE8
respond.min.js - For Media Queries support in IE8

## Getting started
### To run locally:
```
npm install
npm start
```
